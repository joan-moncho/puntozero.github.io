# Punto Zero ✦

> Reserva tu cita. Sin llamadas, sin esperas.

Aplicación web de reservas online para **Punto Zero**, peluquería en Madrid. Desarrollada como una Single Page Application con disponibilidad en tiempo real — cuando un cliente reserva una hora con un estilista, esa franja queda bloqueada automáticamente para el resto de usuarios.

---

## ✦ Funcionalidades

- **Reserva en 3 pasos** — servicio → estilista → fecha y hora
- **Disponibilidad en tiempo real** — los horarios ocupados se bloquean al instante
- **Protección anticolisión** — doble verificación justo antes de confirmar
- **Catálogo de servicios** — con precios y duración estimada
- **Fichas del equipo** — con especialidad y valoraciones
- **Panel de administración** — dashboard, citas, equipo y servicios
